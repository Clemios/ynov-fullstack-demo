# ynov-fullstack-demo

Ce repo a été réalisé pour un cours de dév web Fullstack à Ynov Toulouse.

Le projet construit pas à pas une stack en microservice basée sur NodeJS et Docker.

L'accent est mis sur la compréhension de chaque brique afin de se rendre compte de la complexité d'un from scratch.

Plus tard, un récapitulatif des étapes dans l'ordre sera dépose sur ce repo.
